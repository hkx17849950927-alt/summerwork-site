# 为谁辛苦为谁忙 · 语音听读页

这是一个可直接部署到 Netlify 的静态网站。

## 已实现

- 页面只保留《为谁辛苦为谁忙》
- 从随站点提供的 DOCX 中读取正文并使用浏览器系统音色朗读
- 音色选择、试听和语速选择
- 自动保存听读位置、音色和语速
- 录制本人声音、试听录音和明确授权确认
- 手机端适配

## 本地预览

不要直接双击 `index.html`，因为浏览器会阻止页面读取 DOCX。请通过任意静态文件服务器打开本目录。

## 语音克隆接口

静态网页不能独立训练或托管克隆音色。接入合规服务后，在页面加载 `app.js` 前设置：

```html
<script>window.VOICE_CLONE_ENDPOINT = "/api/voice-clone";</script>
```

接口接收 `multipart/form-data`：

- `audio`: 用户录音
- `consent`: 固定值 `self-voice-confirmed`

返回 JSON：`{ "voiceId": "..." }`。生产环境还应加入身份验证、明确的数据保留/删除策略，并限制只能克隆录音者本人的声音。
